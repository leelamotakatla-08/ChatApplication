# ChatApplication
A web-based messaging application inspired by WhatsApp, developed using HTML, CSS, JavaScript, and Node.js.
This project demonstrates core chat functionalities such as contact management, messaging, and chat controls in a simple and interactive interface.

Features
Contact Management
Add new contacts with name and phone number
Delete contacts using long-press interaction
Messaging System
Send messages in real-time (simulated)
Auto-reply functionality for incoming messages
Chat Controls
Delete individual messages
Clear entire chat history
Mute/unmute chat notifications
User Interface
Clean and structured layout
Easy navigation and interaction
 Tech Stack
Layer	Technology
Frontend	HTML, CSS, JavaScript
Backend	Node.js
